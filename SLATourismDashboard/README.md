# 🌍 Tourism Dashboard

A modern web dashboard for analyzing and visualizing tourism data in Sri Lanka.
This project provides insights into tourist reviews, ratings, locations, and sentiment analysis to support tourism industry decision-making.

---

## 📌 Project Overview

The **Tourism Dashboard** is designed to help analyze tourism trends using interactive visualizations and data analytics.

The system allows users to:

* Explore tourist destinations in Sri Lanka
* Analyze tourist reviews and ratings
* Visualize tourism data using charts and dashboards
* Support tourism stakeholders with data-driven insights

## ⚙️ Installation Guide

### 1️⃣ Clone the repository

```
git clone https://github.com/mathumathusan/TourismDashboard.git
```

---

### 2️⃣ Run the React Frontend

```
cd frontend-react
npm install
npm start
```

Frontend runs at:

```
http://localhost:3000
```

---

### 3️⃣ Run the ASP.NET Backend

Open the backend project in **Visual Studio** and run:

```
dotnet run
```

Backend API runs at:

```
https://localhost:5001
```
